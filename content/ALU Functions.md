# ALU Functions
## X Inputs
- If zx = 1, then x=0
- If nx = 1, then x=!x
## Y Inputs
- If zy = 1, then y=0
- If zy = 0, then y=!y
## + or &
- If f=1, then out=x+y
- If f=0, then out=x&y
## Output
- If no=1, then out=!out
## Type Outputs
- If out=0 then zr=1, else zr=0 (zero)
- If out=0 then ng=1, else ng=0 (negative)
