# Counters
- Can help with telling a computer to execute a list of steps
## Functions
- Resetting the value (out[t+1]=0)
- Incrementing the value (if inc[t]=1, out[t+1]=out[t]+1)
- Skipping to a desired value (if load[t]=1, out[t+1]=in[t])
