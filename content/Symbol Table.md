# Symbol Table
## Process
### First Pass
1. Initialization
	- Adds pre-defined symbols
2. Scan program to look for labels 
	- Adds found labels into symbol table
### Second Pass
1. Set n to 16
3. Add Variable Symbols
	- Create new variable if not found
	- n++
