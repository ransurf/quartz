# Addressing Modes
1. Register
	- ex) R2 = R2 + R1
2. Direct
	- ex) Mem[200] = Mem[200] + R1
3. Indirect (Variables?)
	- ex) Mem[A] = Mem[A] + 1
4. Immediate 
	- ex) R1 = R1 + 73
