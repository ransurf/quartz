# Array vs Linked List
## Principles
Value-oriented

| Operations      | Array-based  | SHSL link-based |
|-----------------|--------------|-----------------|
| getElementCount | 1            | 1               |
| insert          | n (shift)    | O(n)            |
| remove          | n (shift <-) | O(n)            |
| removeAll       | 1            | O(n)            |
| retrieve/get    | O(1)         | O(n)            |
