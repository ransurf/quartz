## Examples
**Declaration and usage**
```c

 // i.e., Hearts = 0, Spades = 1, Clubs = 2, Diamonds = 3;

 enum emphasis {Bold = 1, Italic = 2, Underline = 4}; // can define integer values of the names

Usage:  enum sui card = Spades; // variable of type enum suit
```
