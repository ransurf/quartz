## Tools for Note Taking
1. Something to write with
	- Recording spontaneous ideas should be simple and encouraged
2. A reference management system
	- Collects references and the notes taken during reading
	- Check out zotero
3. A slip box
	- Zettelkasten
4. An editor
	- Grammarly
