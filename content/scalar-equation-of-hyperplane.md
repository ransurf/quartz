## principles
- Given point p and normal n, find a scalar equation
?
- n(x-p) = 0
	- Create equations for each component of vector, x = x1, x2, etc
- nx - np = 0
- nx = np
- Plug in point and normal, solve for ax + by + cz  ... = d type beat
