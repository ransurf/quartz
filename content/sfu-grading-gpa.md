# SFU Grading
Generally for sciences & appsci you want to be in the B+ to A- range

 | Letter Grade | GPA  |
 | ------------ | ---- |
 | A+           | 4.33 |
 | A            | 4.0  |
 | A-           | 3.67 |
 | B+           | 3.33 |
 | B            | 3.0  |
 | B-           | 2.67 |
 | C+           | 2.33 |
 | C            | 2.0  |
 | C-           | 1.67 |
 | D            | 1.0  |
 | F            | 0    |
